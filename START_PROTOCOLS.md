## ⚡ Бързо Начало (5 минути)

### 1️⃣ Добави `.env.local`

Добави тези редове в твоя `.env.local` файл:

```env
GOOGLE_DRIVE_PROTOCOLS_FOLDER_ID=12axMX_3K5-HcFNWm3KcsKlnLOhHM5MW1
NEXT_PUBLIC_ENABLE_PROTOCOLS=true
PROTOCOLS_CACHE_DURATION=48
```

### 2️⃣ Рестартирай Сървъра

```bash
npm run dev
```

### 3️⃣ Тестирай

Отвори: **http://localhost:3001/protocols**

✅ Трябва да видиш списък със протоколи от Google Drive!
